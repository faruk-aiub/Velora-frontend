# Velora Frontend

Velora is a full-featured modern e-commerce frontend application built with **Next.js, React, and TypeScript**.  
The application provides a smooth shopping experience with product browsing, authentication, cart management, checkout flow, and user dashboard features.

---

## 🚀 Features

### User Features

- User registration and login
- Email verification
- Forgot password and reset password
- Secure authentication system
- User profile management
- Browse products
- Search and filter products
- View product details
- Product variants selection
- Add products to cart
- Update cart quantity
- Remove items from cart
- Order placement
- View order history

### Admin Features

- Admin dashboard
- Manage products
- Create, update and delete products
- Manage product categories
- Manage users
- Manage orders
- Upload product images/files
